# Frontend Requirements — NBA Stats

## Дизайн-система

**Стиль:** Monkeytype-inspired минимализм

- **Фон:** `#1a1b1e` (почти чёрный, не чисто чёрный)
- **Карточки/поверхности:** `#2c2e33`
- **Акцент:** `#4ade80` (зелёный)
- **Текст основной:** `#d1d5db`
- **Текст вторичный:** `#6b7280`
- **Текст muted:** `#4b5563`
- **Граница:** `#374151`
- **Шрифт:** `Roboto Mono` (моноширинный, как у monkeytype)
- **Без теней, без градиентов** — плоский матовый стиль
- **Минимум рамок** — разделение через отступы и цвет фона
- **Анимации:** только fade/opacity transitions, без слайдов

## Вкладка: Dashboard

### Фильтр сезона
Глобальный селектор сезона в топбаре — влияет на все графики "по сезону".

### Блок 1 — Топ-5 по PER (выбранный сезон)
- Список из 5 игроков: фото, имя, команда, значение PER
- Клик → открывает модальное окно игрока

### Блок 2 — График по сезонам: Avg PTS команды
- Line chart: ось X = сезоны (2019-20 … 2023-24), ось Y = среднее очков за матч по лиге
- Данные: среднее `avg_pts` по всем `player_season_stats` за каждый сезон

### Блок 3 — График по сезонам: Avg 3P% по лиге
- Line chart: ось X = сезоны, ось Y = среднее `fg3_pct * 100` по лиге
- Показывает рост трёхочковой эры

### Блок 4 — График по выбранному сезону: Топ-10 команд по avg PTS
- Bar chart: горизонтальный, ось Y = команды (abbreviation), ось X = avg PTS
- Данные: среднее `avg_pts` по `player_season_stats` сгруппированное по `team_id`

### Блок 5 — График по выбранному сезону: Топ-10 игроков по USG%
- Bar chart: горизонтальный, ось Y = имя игрока, ось X = USG% (в %)
- Данные: топ-10 из `/stats/leaders?metric=usg_pct&season_id=...&limit=10`

## Модальное окно игрока

Открывается поверх любой страницы (Players, Dashboard, Leaderboard).

### Структура:
**Шапка:**
- Фото игрока (большое)
- Имя, команда (логотип), позиция, номер
- Рост, вес, возраст

**Метрики текущего сезона:**
- Строка бейджей: PTS / REB / AST / STL / BLK / FG% / 3P% / FT%
- Строка бейджей: PER / TS% / eFG% / USG% / BPM

**Радар характеристик:**
- 6 осей: **PTS / REB / AST / PER / BPM / TS%**
  - PTS — результативность (атака)
  - REB — подборы (борьба за мяч)
  - AST — плеймейкинг
  - PER — общая эффективность (интегральная метрика)
  - BPM — вклад относительно среднего по лиге (контекстуальная)
  - TS% — эффективность бросков (качество атаки)
- Значения нормализованы относительно максимума по лиге в данном сезоне (0–1)
- Библиотека: Recharts `RadarChart`

**Кнопка "Добавить к сравнению":**
- Добавляет игрока в глобальный стейт сравнения (максимум 2 игрока)
- Если уже 2 игрока выбраны — кнопка "Сравнить" открывает модал сравнения

**Game Log:**
- Таблица последних 20 матчей (как сейчас)

---

## Модальное окно сравнения игроков

Открывается поверх модала игрока или через отдельную кнопку.

### Структура:
- Два столбца: игрок 1 vs игрок 2
- Шапки обоих игроков (фото, имя, команда)
- **Совмещённый радар** — два полигона на одном RadarChart (разные цвета)
- Таблица метрик: PTS / REB / AST / PER / BPM / TS% / eFG% / USG%
  - Выделяется зелёным лучшее значение в каждой строке
- Кнопка закрыть

## Глобальный стейт сравнения

- Хранит массив из 0-2 игроков (`player_id`)
- Плавающая кнопка в правом нижнем углу — появляется когда выбран хотя бы 1 игрок
- Показывает мини-аватары выбранных игроков и кнопку "Сравнить" (активна при 2 игроках)
- Кнопка "×" для сброса каждого игрока
