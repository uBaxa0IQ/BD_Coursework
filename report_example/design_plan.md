  - Модуль трассировки лучей

## 2.2. Математическое описание (или Описание алгоритмов)

- 2.2.1. Математические модели и формулы:
  - Модель освещения (Закон Ламберта):
    - Диффузное отражение для матовых поверхностей
    - Формула: I = I₀ · ρ · max(0, n⃗ · (-d⃗))
    - Обозначения переменных
    - Возможность предварительного расчета и сохранения в текстуру
  - Преломление (Закон Снеллиуса):
    - Формула: n₁ sin(θ₁) = n₂ sin(θ₂)
    - Перераспределение лучей и формирование световых узоров
  - Моделирование матовости (рассеивание):
    - Стохастическое рассеивание для имитации матовых материалов
  - Методы распределения лучей:
    - Равномерное распределение на основе сферической решетки Фибоначчи
    - Формулы для координат точек на сфере
    - Золотой угол φ = π(3 - √5)
    - Преимущества равномерного распределения без кластеризации
    - Стратифицированная сетка (grid sampling) как альтернатива

- 2.2.2. Описание структур данных:
  - Представление источников света (позиция, интенсивность, цвет, количество лучей)
  - Представление объектов сцены (тип, геометрия, материал)
  - Представление защитных колпачков (внутренний и внешний радиус, коэффициент преломления, матовость, цветовой фильтр)
  - Представление камеры (позиция, направление, разрешение, поле зрения)
  - Представление поверхностей (стена, сферы) и их карт освещенности

- 2.2.3. Блок-схемы основных алгоритмов:
  - Алгоритм предварительного расчета освещения:
  - Испускание лучей от источников света
  - Обработка защитных колпачков (преломление)
  - Трассировка лучей до поверхностей
  - Накопление света по модели Ламберта
  - Формирование карт освещенности
  - Алгоритм трассировки луча:
  - Проверка пересечений с объектами сцены
  - Определение ближайшего пересечения
  - Расчет интенсивности по модели Ламберта
  - Запись в буфер освещенности
  - Алгоритм обработки преломления в колпачке:
  - Определение входа в внутреннюю сферу (воздух → материал)
  - Расчет преломления на первой границе
  - Определение выхода из внешней сферы (материал → воздух)
  - Расчет преломления на второй границе
  - Обработка полного внутреннего отражения
  - Применение поглощения света
  - Применение цветового фильтра
  - Рассеивание направления луча
  - Алгоритм визуализации вида камеры:
  - Генерация лучей от камеры в направлении сцены
  - Проверка пересечений с объектами
  - Определение координат на поверхности
  - Сэмплирование из карты освещенности
  - Суперсэмплирование для сглаживания
  - Формирование итогового изображения
  - Алгоритм распределения направлений лучей:
  - Стратифицированная сетка на сфере
  - Равномерное распределение по сфере Фибоначчи
  - Генерация единичных векторов направлений
  - Двухэтапный рендеринг с предварительным расчетом:
    - Этап 1: Предварительный расчет освещения (baking)
    - Этап 2: Визуализация
