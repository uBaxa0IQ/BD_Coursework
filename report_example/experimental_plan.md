# План исследовательской части

## 4.1. Методика тестирования

- Как вы проверяли программу:
  - Черный ящик: тестирование функциональности без знания внутренней реализации
  - Белый ящик: тестирование внутренней логики алгоритмов
  - Нагрузочное тестирование: проверка производительности при различных параметрах
- Характеристики компьютера, на котором проводились тесты:
  - Процессор
  - Оперативная память
  - Операционная система
  - Версии используемого ПО
- Методика измерения времени выполнения:
  - Измерение суммарного времени выполнения функций `bake_lighting()` и `render_view_only()` с использованием высокоточных таймеров (`time.perf_counter()`)
  - Количество прогонов для усреднения результатов (минимум 3 прогона для каждого измерения)
- Параметры тестовых сцен:
  - Базовые параметры для всех экспериментов
  - Вариации параметров для каждого эксперимента

## 4.2. Результаты тестирования

- 4.2.1. Примеры работы программы на разных данных:
  - Корректные данные:
    - Различные конфигурации источников света
    - Различные параметры колпачков
    - Различные геометрии сцен
  - Некорректные данные:
    - Обработка ошибок при некорректных входных данных
    - Поведение системы при граничных значениях

- 4.2.2. Графики производительности (Зависимость времени работы от объема данных):
  - Эксперимент 1: Зависимость суммарного времени расчета от количества источников света
    - Описание эксперимента: измерение суммарного времени расчета (запекание + отрисовка) при различном количестве источников света
    - Параметры эксперимента:
      - Количество источников света: 1, 5, 10, 20, 50, 100
      - Фиксированное количество лучей на источник: 200
      - Фиксированное разрешение карт освещенности: 600×600
      - Отсутствие объектов сцены (только стена)
      - Одинаковые параметры всех источников
    - Результаты: таблица и график зависимости суммарного времени расчета от количества источников света
    - Анализ результатов: определение характера зависимости (линейная, квадратичная и т.д.)
  
  - Эксперимент 2: Зависимость суммарного времени расчета от количества объектов сцены
    - Описание эксперимента: измерение суммарного времени расчета при различном количестве сфер в сцене
    - Параметры эксперимента:
      - Количество объектов (сфер): 0, 2, 5, 10, 15, 20
      - Фиксированное количество источников света: 10
      - Фиксированное количество лучей на источник: 200
      - Фиксированное разрешение карт освещенности: 600×600
    - Результаты: таблица и график зависимости суммарного времени расчета от количества объектов
    - Анализ результатов: оценка влияния проверки пересечений на производительность
  
  - Эксперимент 3: Зависимость суммарного времени расчета от разрешения карт освещенности
    - Описание эксперимента: измерение суммарного времени расчета при различных разрешениях карт освещенности
    - Параметры эксперимента:
      - Разрешения: 200×200, 400×400, 600×600, 800×800, 1000×1000
      - Фиксированное количество источников света: 10
      - Фиксированное количество лучей на источник: 200
      - Отсутствие объектов сцены (только стена)
    - Результаты: таблица и график зависимости суммарного времени расчета от разрешения
    - Анализ результатов: оценка влияния размера буферов на производительность
