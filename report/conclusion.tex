\chapter*{ЗАКЛЮЧЕНИЕ}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}

В ходе выполнения курсовой работы была успешно спроектирована и реализована база данных для хранения и анализа статистики игроков и команд NBA, а также разработано клиент-серверное программное обеспечение для взаимодействия с ней.

Все поставленные задачи были выполнены в полном объеме:
\begin{enumerate}
	\item Проведен анализ предметной области, выделены основные сущности (игроки, команды, матчи, статистика) и формализованы правила расчета продвинутых метрик (TS\%, eFG\%, PER, BPM).
	\item Спроектирована реляционная база данных, состоящая из 8 таблиц. Доказано соответствие структуры базы данных требованиям третьей нормальной формы (3НФ).
	\item База данных успешно развернута под управлением СУБД PostgreSQL и наполнена репрезентативным объемом данных (около 200 тысяч записей), что многократно превышает минимальные требования.
	\item На стороне СУБД (на языке PL/pgSQL) реализованы хранимые функции для вычисления аналитических показателей, что позволило оптимизировать передачу данных по сети.
	\item Разработана и внедрена ролевая модель доступа (администратор, аналитик, читатель), обеспечивающая безопасность данных.
	\item Разработано современное веб-приложение (Backend на Python/FastAPI, Frontend на React/TypeScript), предоставляющее удобный пользовательский интерфейс для просмотра дашбордов и статистики.
	\item Внедрен механизм кэширования на базе InMemory СУБД (Redis), обеспечивающий быстрое обслуживание ресурсоемких запросов.
\end{enumerate}

В исследовательском разделе был проведен практический эксперимент по оценке производительности системы. Результаты показали, что применение составных индексов (B-Tree), учитывающих условия фильтрации и сортировки, сокращает время выполнения сложных выборок более чем в 2 раза. В свою очередь, использование кэширования Redis позволило ускорить время ответа API в 4,7 раза и обеспечить стабильную работу системы (0 ошибок) при конкурентной нагрузке до 100 параллельных запросов.

Полученные результаты подтверждают эффективность выбранных архитектурных решений и технологий для создания высоконагруженных аналитических систем в сфере спортивной статистики. Разработанный программный комплекс может быть использован в качестве основы для создания профессиональных аналитических платформ.
